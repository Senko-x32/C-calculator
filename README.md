
# Calulator in C

This is a calulator made in C that can loop back to the menu after giving you your results.


## How to run
You first want to make sure you have gcc installed so run the following command.


```http
  apt install gcc
```
Then you want to compile the C code so you can run the program.


```http
  gcc main.c -o calc
```
Now you can run it by using the following command.


```http
  ./calc
```
